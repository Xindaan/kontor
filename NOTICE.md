# Data sources and third-party licensing

The MIT licence in `LICENSE` covers the **code** in this repository. It does not
and cannot cover the third-party data that ships alongside it: that data was
produced by other people under other terms. This file lists what is bundled,
where it came from, and what that means if you reuse it.

If you only run the code against your own data, none of this applies to you.

## Market data pulled at runtime

The loader fetches prices, dividends, analyst data and news from **Yahoo
Finance** (via `yfinance`) and optionally **Finnhub** and **NewsAPI**. That data
is not redistributed here — it is downloaded into your local cache when you run
a backtest, and it is governed by each provider's terms, not by this repository's
licence. Vendor terms generally permit personal and research use and restrict
redistribution; check them before you republish anything derived from a run.

## If you are reusing the constituent data

The point-in-time membership files are the ones with a genuine share-alike
obligation attached. The cleanest route is to regenerate them rather than copy
them from here:

```bash
python scripts/download_pit_data.py --index sp500
```

That way your copy carries whatever attribution your own project needs, and you
get the current state of the upstream articles rather than a snapshot of them.
